## 执行指令

1.  **阅读**: 打开当前会话中最近创建的 `specs/[feature_name]/plan.md`。
2.  **执行操作**:
    * 定位到包含复选框的 **分步实施 (Step-by-Step)** 章节。
    * 按顺序执行所有未选中的项目 `- [ ]`。
    * **语言要求**: 执行过程中给出的任何解释、日志或反馈必须使用**简体中文**。
    * **并行优化**: 如果多个步骤之间无依赖关系，可以同时执行以提高效率。
3.  **编译验证（强制）**:
    * 代码修改步骤全部完成后，**必须**执行 `npm run build` 确认编译通过。
    * 如果编译失败，立即修复错误并重新编译，直到通过为止。
4.  **状态同步**:
    * 每个步骤完成后，将 `plan.md` 中对应的状态从 `- [ ]` 更改为 `- [x]`。
    * 所有步骤执行完毕后，统一更新 plan.md（可批量标记）。
5.  **完成汇报**:
    * 以表格形式汇总所有步骤的完成状态。
    * 提醒用户进行验证测试。
