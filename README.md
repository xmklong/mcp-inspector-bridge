# MCP Inspector Bridge

`mcp-inspector-bridge` 是专为 **Cocos Creator 2.4.x** 打造的现代化运行节点审查与 DevTools 桥接插件。
本项目旨在原版 `CocosInspector` 机制的基础上，提供更可靠、更先进的双分栏调试体验，并彻底解决内嵌 Chromium DevTools 时常见的挂起死锁问题。

## ✨ 核心特性

- **🚀 双分栏工作流 (Dual-Pane Layout)**
  采用现代化的 Vue 3 构建插件主面板。左侧保留原生 Webview 渲染游戏视口，右侧原生集成 Chrome DevTools，实现“边玩边审”的沉浸式体验。
  
- **🛡️ 稳态 BrowserView 底层架构与极净防崩壁垒 (Anti-Crash Barrier)**
  彻底规避了 `webContents.setDevToolsWebContents()` 在 `<webview>` 标签上的 `about:blank` 导航死锁问题。通过逆向深入还原，采用 Electron 原生 `BrowserView` 作为 DevTools 容器，确立了绝对稳固的 CDP (Chrome DevTools Protocol) 链路。
  - **全隔离的场景存活校验网 (Event-Driven Scene Sandbox)**：痛下决心移除了旧版激进但致命的底层网络抢跑扫描（尤其是在启动期盲目 `fetch` 端口 `settings.js` 的越位探测）以及过于自负的初始预设渲染。通过封装隔离出全新的 `initializePreviewEnvironment` 沙盒并确立以编辑器特权级别 IPC `isEditorSceneActive` 为不可越界的防御底牌：使插件挂载起手保持为完全不作为的保守等待态 (`false`)。只有真正拿到主进程返回的回调安全判定，插件才会首次启动嗅探并释放面板渲染指令 `refreshGame`，从根源上绝对击杀了由于编辑器为空场景或未就绪时被唤醒探测导致引擎触发 `stashScene` 而抛出 `TypeError` 满盘崩溃死锁。
  - **单次唤醒与后台场景静默剥离**：引入了防抖锁闭包状态保护（如 `isEnvInitialized` 与 `hasInitialRefreshed`）。系统在整个生命周期内部只要确认完成了最沉重的那一次初启端口嗅探及加载，后续所有编辑器端的切场景、失焦、短时扰动都会被智能规避隔离。不仅杜绝了往昔极其生硬的频繁重起与画面黑屏，你的游戏运行场面更将丝滑地挂起留在大屏视野，再也不会被迫跟着开发流程重置！
  - **降级轮询自控销毁 (Fallback Self-Dismiss)**：原生 IPC 失联迫降为 DOM 心跳包预案后，浮窗警告采取与事件解绑的闭包自律生命周期机制。在履约 2 秒明示告知后静默遁回，不侵占一分一毫宝贵的纵向开发视野与屏幕红利。
  - **多实例多开端口防御 (Multi-Instance Tolerant)**：考虑到游戏多项目并行研发的环境，全面增强优化了对于预览端 `localhost:` 的自适应 URL 包容规则。不再执拗硬编码 `7456`，通过深度挖掘引擎私有变量 `_previewPort` 配合前端物理级的网络嗅探轮询探针（`probeAlivePort`）构成极其强悍的双重火力，彻底击穿并校准了多开编辑器时端口被自增抢占而导致的黑屏与串联画面顽疾。
  - **全形态窗口适配与保活 (Standalone & Docked)**：无论是内嵌在编辑器面板中，还是作为“独立窗口”悬浮，插件皆能秒级重连唤醒。针对单独弹出的开发者工具窗口，创新性采取了原生 `BrowserWindow` 手动句柄捕获与物理隐藏 (`hide()/show()`) 方案，实现在工作面板标签切换时的秒级切后台与“隐身”，维持全部断点、抓包等前置上下文，提供永不断档即切即用的绝杀调试体验。

- **⚡ 智能运行时探针与节点树架构 (Runtime Probe & Node Tree)**
  基于预加载脚本 (`preload.js`) 动态无侵入地向游戏运行时注入核心探针。不仅能瞬时截获游戏 DOM 树，更深度适配了 Cocos Creator 2.4 引擎底层的属性陷阱（完美跨越 `cc.Scene` 对 `active` 的 getter 报错劫持）。
  - **高级组合与穿透搜索引擎 (Deep Search Match)**：右槽面板支持强大的节点搜查定位机制。除基础的名称正则过滤及一次性分词高亮外，现已全面支持通过**空格切分词缀**的跨域多点交叉匹配（AND 逻辑）；不仅限于节点名本身，更能**原生穿透深达组件的装配层级**进行隐性类名的截获搜查（诸如只输入 `Animation` 即可揪出所有挂载了该行为的素干对象），并在判定命中而未受词根显性影响下自发提供高贵的界面级灰色副文本 `(cc.Animation)` 直列标注。
  - **严格路径净化锁定 (Strict Path Filtering)**：摒弃了原始生拉硬拽的扁平化匹配展现。现在启动搜索时，算法利用深度优先策略打穿全节点森林，实现「仅展示命中节点本身」外加「串连其生存的历代高悬直系长辈」呈现。至于其他丝毫没被触碰到的偏支分系、连带无匹配项的平级子嗣，将被无情剔除彻底隐藏。一发搜索即可在繁杂乱序的海量实装中为你开辟一条极其纯粹、没有杂散噪点的绝对坐标航线。此外，一键隐去搜索词时系统会依靠 `ancestorIds` 在瞬息保留原路径祖先的折叠位点，避免长篇大论的树状列表遭遇暴力重弹和视线丢失。
  - **可置顶的所见即所得节点高亮排查器 (Anti-ZFighting Highlight Overlay)**：无需依靠死板猜测！随着鼠标掠过树状图，对应目标节点（无论是宽带为0的点阵还是藏在地下的死角）皆能在编辑器的真实运行画面内浮现出精良渲染的、永不穿模被盖的专属金/蓝高光轮廓。更绝的是背后的“双轨盗视画中画”管线：探针主动搜刮全员摄影机顶端的图层组号加以掩护变装，完美破解因为混合 UI/3D 群组而引发的光效遮挡吞噬难题。
  - **实况屏幕节点拾取器 (On-Screen Node Picker)**：告别繁琐的节点树翻找！直接在游戏预览画面中启用十字准星拾取模式。它内建了极度敏锐的 **候选池面积穿透权重算法 (Area-Weighted Raycast)**，即便复杂的层叠 UI 中存在尺寸巨大的全屏透明遮盖层（Mask / ClickGuard），算法也会自发剥离这些宽泛巨物，精准“穿透”并优先捕获您真正意图点选的微小按钮与文字控件。为了解决各异的 Canvas `Fit` 缩放策略与多重嵌套 `Viewport` 相机带来的坐标矩阵测算盲区，底层探针劫持并全盘接引了黑盒引擎内部最权威的原生态 `node._hitTest` 点击事件管道。无论是怎样的魔幻分辨率偏移、层级缩放，准星只需对标视觉原生落点，便能实现指哪点哪的像素级命中。此外，拾取器自带**最高图层原生物理游标探测器 (DOM Tracker) **以根绝游戏黑边偏差。拾取成功后，左侧的节点结构树会全自动越级伸展（跨越十几层深不见底的嵌套）、顺滑滚跃至视线中央；连同持久的焦点高亮边框驻留，体验碾压一众传统常规方案。
- **🔍 组件属性检查器 (Node Inspector)**
  选中实时树节点后，完美复现了引擎原生层级的 Inspector。
  - **组件数据层级遍历与 JSON 控制台一键导出**：在每个组件头部挂载轻量级的 🖨️ 打印输出模块。探针底层深度拦截了引擎级别的 `cc.Node`、`cc.Asset` 等极易导致序列化死锁的复杂实例，降维转化为高可读性的 `[ cc.Node: path/to/name ]` 具象化结构树；同时搭载原生 `WeakSet` 防御网彻底根治所有由于对象互相挂载而诱发的单层数据回环（Circular Reference）崩溃，为大型游戏业务数据对象提供极度安全便捷的代码审查控制台直出。
  - **组件通用启停管控**：为所有组件引入层级通用的 `enabled` 开关，实现组件运行沙盒期的一键强行休眠与防报错重排版唤醒。
  - **纯运行时级无痕注入**：所有增删改查交互指令直接通过 WebContents JavaScript 原生注入，**绝对不脏化原编辑器 Scene 数据（彻底告别退出保存弹窗阻断）**。
  - **基础类型跨屏双向修改**：打通 `number`, `string`, `boolean` 型属性在 Vue UI 层和原生游戏运行实例层级的实时修改通道。
  - **全景节点基础属性覆盖**：深度补齐了对齐原生编辑器的 `Anchor`（锚点）、`Color`（带有反向 Hex 解析的安全颜色选择器）、`Opacity`（带越界钳制的透明通道）、`Skew`（倾斜角）以及 `Group`（支持底层引擎枚举字典即需拾取的渲染群组下拉切换）。实现对节点核心 Transform 参数 100% 同步可视化双向修改监控。
  - **精准过滤引擎私有属性**：深入 `__props__` 和 `__attrs__` 数据流，完美识别 `@property({ visible: false })` 以及下划线开头私有变量，只呈现允许暴露给用户的安全属性。
  - **核心组件追踪与越级定位 (Reference Locator)**：属性检查器现已全面引入对象与资产的精准定位（`🎯`）响应枢纽。不仅针对一级节点 (`node_ref`) 或各类直接引用的素材资源 (`asset_ref`) 提供一键呼出层级树折叠定位和资产闪烁检索的强力追踪；更能**深度穿透引擎原生的 `Array` 封装数据源**，将长阵列中的每一个零碎子实体都提取转化为对应的专属类型色板展示及悬浮快定器，无惧深层重叠数据流的隐蔽。
  - **智能化枚举控件衍生 (Dropdowns)**：自带深层数据挖掘补丁。针对诸如 `sp.Skeleton` 等具有复杂内部资源的组件，爬虫能够穿透解构其挂载的运行时数据，自动抽离出所有可用的 `animations` 和 `skins` 列表。到达 Vue 渲染端后自动将传统的无约束字符串文本框升格为原生下拉选框（`<select>`），绝不留给用户手动拼写出错的机会。
  - **注入态临时日志控制器**：自带防止刷屏的黑客级浮动调试框，一键开关，用于深层排查由于隐去某些属性导致的显示偏差。
  - **动态向后兼容 (Deprecated Feature Safety)**：自动抹平版本引擎迭代中触发的 `cc.Node.rotation` 黄字废弃警告风暴，依靠向后降级的 `-angle` 倒置换接平缓修复各个节点的方向表现而由于爬虫被独立层拦截，真正达成无死角适配且零侵入体验。
  - **统一的极净属性分发管线 (Unified Property API)**：全盘抛弃在早年由面板通过零散 `executeJavaScript` 拼接修改底层内存对象的“意大利面条式”后备乱码代码。利用直接装配的防空指针探针层方法 `window.__mcpCrawler.updateNodeProperty` 实现 0 报错、无延宕且自带布局刷新缓冲 (`updateAlignment`) 的高级防损双向绑定架构。
- **📉 内存全域检视与溯源查表 (Memory Profiler & Deep Asset Tracing)**
  具备无视底层编译压缩的透骨查视能力。不局限于只抓取内存大小，更能按引擎的 `Bundle` 管辖区域对成千上万细碎的内存颗粒进行分门归类与全局排行！
  - **全貌追踪级水位图与热传感**：面板在接收侧开辟了独立的记录维度，无论是随着运行更迭还是资源卸废，都会准确为你截留下各个区域乃至历史以来的极限高点 `Max` 和谷点 `Min`。它内置随风而动的趋势标识（↑↓），令潜逃中的内存暴涨事件和流出无所遁形。
  - **混淆逆向解离与 Editor 私有映射**：彻底破解 Cocos 打包或资源管理流程中所引起的深度 `UUID` 乱码风暴与名录丢失。搭载着从 `SpriteFrame` 逆推 `Texture2D` 所有权的侦察机制以及自愈级多路拦截截断法，最后在安全端大举召唤最高特权级别的 `Editor.assetdb.remote.uuidToUrl` 等 API 进行强行解码；将原本晦涩丑陋的长串识别序列和满屏无意义的 `[Unnamed]` 彻底清洗转换为初见原版那般清白爽朗的 `textures/hero_walk.png` 文件级真实路径展示体系。
  - **全息资源反向导航网 (Asset Manager Quick Locator)**：打破分析面板和底层编辑器的数据壁垒，在所有解构出的物理与 Bundle 资源旁提供一键定位交互枢纽（`🎯`）。轻触即可利用无缝融合的底层特权 IPC 唤醒引擎资源管理器并高亮选中资源实体文件。附带内置前缀防越界拦截锁（自动屏蔽内建运行时虚拟纹理与无效缓存），在实现“边看表单锁定内存泄漏，边删文件源”的高效追击闭环之余，彻底清除了向编辑器投递空位 IPC 引发悬挂和卡顿的风险。
- **⏭️ 引擎时钟与单步控制 (Clock & Step Control)**
  全局单帧步进与暂停状态的双向同步。无论是通过控制面板下发暂停指令，还是监听引擎底层的 `cc.game` 状态变化，面板都能实时动态切换“⏸ 暂停”与“▶️ 播放”的视图表现。在运行时游戏点击“单帧运行”，将自动拦截定格引擎并精确推演下一帧。

- **🎛️ 灵活的窗口分辨率与界面的自适应折叠 (Responsive UI)**
  支持横/竖屏动态切换、自定义多端尺寸模拟，无缝处理 BrowserView 在面板缩放和重布局时的动态占位叠加（DOM Rect 同步）。
  - **沉浸式并排视窗与无缝侧旋分割线 (Horizontal Dual-Pane Divider)**：颠覆了传统面板“上树下属性”纵向拥簇的憋屈态势，将主体审查模块重组为左（节点层级）、右（组件清单）并行的开阔 IDE 排版！中缝嵌入全时段响应的可调焦隔断器（Resizer），拖动极其如丝顺滑、彻底摆脱框架吸附跳闪，完美守护最低视野界限并搭载全自动存档体系。
  - **标签页自选拖拽重排与固化 (Draggable Tabs)**：打破死板的固定视图结构，基于底层原生 HTML5 能力赋予了顶部工具标签页丝滑的拖拽排序交互。配有蓝色占位插入指示器与拖延影像截取保护。任何习惯性的位移重摆均会被瞬间记录并击穿存储至宿主本地凭证，无论经过多少次系统重启乃至全盘重载，都能自动依据私有定制习惯完美复位展示布局。
  - **优雅的小窗折叠降级**：当用户将面板宽度极致压缩时，顶部操作栏会自动退化至“Icon Only”全图标模式，其余辅助文本主动开启省略保护。确保任何极端拉扯皆不破版发生两栏畸形折行。右侧的所有标签与下方的属性检查面板亦实现极致抗压收缩和原生边界事件保护抗体，不留一丝框外重叠的破绽！
  - **防溢出与沉浸式画面**：底层严格处理了各种异形窗口及极端缩放带来的 DOM 亚像素（Sub-pixel）运算精度误差，并通过 Webview 拦截器动态注入断绝一切原生滚动条展现的强力 CSS 锁，保障画面边界100%纯净。
  - **无缝平滑越狱与防漂移系统**：彻底攻克了传统通过 `clientX` 全局定长捕获减法公式在嵌入编辑器分槽内定位不准的历史顽疾。新一代的手柄增量拉扯架构即使身处重重叠加的 Cocos 巢套内深，一样保持像素级的神仙随动跟踪。

- **🩺 全景渲染调试与断流诊断器 (Frame & Batch Debugger)**
  专门斩断 DrawCall 暴涨与性能黑盒痛点，实现在运行时环境内针对场景合批状态的活体解剖。不需要靠猜测和排查，插件会直接指出毁掉整个队列的“罪魁祸首”。
  - **零污染静默拦截与时间轴顺序重位法 (Array Shift Sequence Strategy)**：直抵渲染底层管线，不仅在 `RenderComponent._checkBatch` 展开预收集，更使用 AOP 在最底层的 `_flush` 分发族系进行强行截断闭箱（Bucket Seal）。依托按帧按序逐次 `.shift()` 的高级游标消费算法，即便 Cocos 引擎为了性能将实际组件全部蒙在极其难以剖绘的虚拟容器 (Dummy New Node) 内部发送给显卡，探针依旧能凭借精确的生命周期同步将真实的材质和组件节点强行翻出，与发往 GPU 的每一条独立物理 DrawCall 做到 1:1 绝对映射捆绑！全程依托原生 `ipcRenderer` 的安全信通传输断批战报，绝对不刷屏污染游戏本属于开发者的原生 `console.warn` 控制台纯净度体验。
  - **全貌聚合的诊断大屏与热核引擎**：通过三位一体的 Hash 分析矩阵（囊括肇事节点、连累的受害节点与具体原因），能够瞬间将庞大的 60FPS 连环断流事故压缩为面板里一条不断跳动的高频徽章，不再发生列表暴涨让内存崩塌卡死现象。
  - **沉浸式三栏全景快照 (Triple-Pane Layout)**：在遭遇复杂场景时，一键“截流当前帧”进入定格分析状态。面板瞬间由断流列表展开为专业的“左(渲染队列) / 中(单步画布) / 右(参数明细)”三栏结构。
  - **多级渲染命令树 (Hierarchical Command Tree)**：左侧自适应呈报该帧由大及小的真实渲染梯队（物理 `DrawCall` -> 引擎逻辑 `RenderCommand`）。结合图文标识，哪一层塞入了哪些组件数据（如 Sprite、Label 判断）一目了然。
  - **逆向探针离屏单步回绘 (Offscreen Step Replay)**：深度劫持底层 `device.draw` 屏障。当你在左侧命令树上下游走时，探针利用高速 IPC 动能不断驱动 Cocos 内核重构缓存截断绘制，并将真实的屏幕显存结果传回中间画布。像剥洋葱一般为你复刻游戏是如何一阶阶把世界画满的。
  - **管线双解与反向追踪 (State Detail & Pinpointing)**：右侧动态呈现当前队列游标所处的深度参数明细（具备智能 WebGL 枚举字典直译的 `Blend` 模式参数、缓冲索引总计、关联材质 Hash），并支持秒级穿透 `[📌]` 回游戏节点的逆向定位能力。

- **💾 面板偏好持久化 (Preference Auto-Save)**
  所有高频核心预览调试参数（包括跨屏断点级的设备分辨率仿真选用、运行时引擎覆盖模块的常显反馈状态、全局免引擎干预级 Webview 音频阻断闸门，乃至右侧工作检查区域的自定推拉排版宽度等）均全盘依托并打通了原生的 `Editor.Profile` 构建端安全体系。这些被强化的选项配置精准落盘于项目沙盒的 `settings` 专用目录下；面板启用的初始间隙瞬间完成逆向提取、Clamp 钳制纠偏与状态回填，一次配置，终身定格，再无任何项目重载初始化时的复原阵痛。

- **🧼 纯净无痕的调试流保障 (Zero-Noise Tuning Experience)**
  全面清除了底层 IPC 通信及 `BrowserView` 初始化注入时产生的大量测试日志、警告以及堆栈跟踪。将所有从核心探针至 UI 映射过程的数据交互做到了极简的静默化抽象，不仅粉碎了原有的刷屏化 Debug 留存，更为开发者自己游戏的控制台预留出了 100% 极净纯洁的排查视野，将插件底层强工具属性带给业务的“干扰感”进行了无死角的原子级抹除。

## 📦 项目结构

```text
mcp-inspector-bridge/
├── package.json          # 插件清单与命令定义
├── main.js               # Cocos 插件主进程入口
├── dist/                 # 编译后的产物
├── src/
│   ├── panel/
│   │   ├── index.ts      # 插件渲染进程，Vue 3 双栏逻辑核心（负责挂载 BrowserView）
│   │   └── index.html    # 插件主面板视图骨架
│   ├── probe/            # 探针模块（被 esbuild 打包为单一 IIFE 闭包注入游戏）
│   │   ├── index.ts      # 探针主生命周期聚合入口
│   │   ├── crawler.ts    # 实时节点树与属性属性爬虫
│   │   ├── highlighter.ts# 悬停/选中高亮渲染遮罩层
│   │   ├── profiler.ts   # 帧绿与逻辑耗时截获测试器
│   │   ├── memory.ts     # 内存全域检视与反向溯源
│   │   ├── render-debugger.ts # 渲染流水线劫持与断批诊断器
│   │   └── picker.ts     # 深度屏幕射线穿透拾取器
│   └── preload.ts        # Electron 预加载中转枢纽，打通 IPC
```

```bash
# 新装依赖 (包含 esbuild)

```
