2. **拾取层（Camera-First Picker）**：反转遍历逻辑。按相机 `depth` 降序遍历所有相机，从每个相机发出射线，**严格校验节点的 `groupIndex` 是否属于该相机的 `cullingMask`**，并手动用 `convertToNodeSpaceAR` 做本地包围盒碰撞。

```javascript
inspectorCamera.orthoSize = visibleSize.height / 2;

    const screenPt = targetCamera.getWorldToScreenPoint(worldCorners[i]);
    drawPoints.push(screenPt);
}

// 3. 画笔锚点已经是 0,0 且没有任何缩放，直接使用屏幕坐标画线！绝对无偏移！
graphics.moveTo(drawPoints[0].x, drawPoints[0].y);
graphics.lineTo(drawPoints[1].x, drawPoints[1].y);
graphics.lineTo(drawPoints[2].x, drawPoints[2].y);
graphics.lineTo(drawPoints[3].x, drawPoints[3].y);
graphics.close();
graphics.stroke();
graphics.fill();
```
